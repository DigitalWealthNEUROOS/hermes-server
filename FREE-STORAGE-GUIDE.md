# GitHub Free Storage - Complete Guide

## Repository Storage
- Repository size: Unlimited* (soft limit 100GB)
- Git LFS: 1GB storage + 1GB bandwidth/month
- GitHub Packages: 500MB
- GitHub Artifacts: 500MB
- GitHub Pages: 1GB

## Codespaces
- Storage: 32GB per codespace
- Free hours: 60h/month (2-core), 30h/month (4-core)
- Prebuilds: Free

## Actions
- Minutes: 2000/month (public), 30/month (private)
- Artifact storage: 500MB
- Cache: 10GB per repository

## External Free Storage
- Cloudflare R2: 10GB + /usr/bin/bash egress
- Backblaze B2: 10GB + 1GB/day download
- Storj DCS: 15GB/month
- MEGA.nz: 20GB
- Google Drive: 15GB

## Total Free Storage: ~200GB+

## Strategy
1. GitHub: Code and small files
2. Git LFS: Large files (models, datasets)
3. Cloudflare R2: Backups and file storage
4. Device: Primary working storage (90GB free)
5. Codespace: Compute only
